# Rise — Daily Tracker ⚡

A premium, dark-themed, atomic habit tracker and daily reflection log designed to build discipline and momentum.

## 🌟 Key Features

* **Score Weighting Engine:** Scored habits sum up to 100% score calculations dynamically each day.
* **Daily Reflection Logs:** Record metrics such as Weight (kg), Mood (1-10), Confidence (1-10), Wins, Weaknesses, and raw thoughts.
* **Interactive Stats Heatmap:** A visual contribution grid displaying habit consistency. Clicking any past/present day slides up a detailed summary card showing:
  * Overall calculated day score.
  * Checklist items completed.
  * Saved log metrics and text entries.
* **Firebase Cloud Firestore Sync:** Keeps your data safely persisted in the cloud.
* **Robust Offline Support:** Runs off-grid and caches inputs locally on mobile, automatically syncing changes to Firestore when connection is restored.
* **PWA Installable:** Runs full-screen directly from your phone's Home Screen without address bars.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables, modern glassmorphism aesthetic), Vanilla JS (ES6).
* **Database:** Firebase Cloud Firestore (Compatibility SDK v10.8.0).
* **Hosting:** Optimized for GitHub Pages or Netlify.

---

## 📱 How to Use on Your Phone

1. **Host it online** for free using **Netlify** or **GitHub Pages**.
2. Open your hosted live link in your phone's browser.
3. **For iOS (Safari):** Tap the **Share** button and select **Add to Home Screen**.
4. **For Android (Chrome):** Tap the **three-dots menu** and select **Add to Home screen** (or *Install App*).
5. Open the newly added app from your home screen and start tracking!
